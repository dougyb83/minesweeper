How to play

The object of the game is to uncover all the tiles that dont have a Mine underneath. Start by clicking any random tile. If you click on a tile with a Mine below it, you have lost the game. If you click a tile with a blank space below, that tile will be removed as well as any tiles surrounding it that have a number below them. If there are more blank tiles next to the one you clicked all the connected blank tiles will also be removed. When you see a number, that number represents how many Mines are surrounding that square. When you think you have identified a Mine Right Click on the tile to place a flag on it. Start by clicking any random tile.
