How to play

The object of the game is to uncover all the tiles that do not have a mine underneath.
Start by clicking any random tile - If you click on a mine, it's GAME OVER.
If you click a tile with a blank space below, that tile will be removed as well as any tiles surrounding it that have a number below them.
If there are more blank tiles next to the one you clicked all the connected blank tiles will also be removed.
When you see a number, that number represents how many Mines are surrounding that square.
When you think you have identified a mine Right Click on the tile to place a flag on it.
